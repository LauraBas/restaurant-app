# Excercice for the Bootcamp F5

### Create a Api in laravel with the frontend in Vue

## Technical Requirements

-   Laravel 8
-   PHP 7.4
-   composer 1.10.15

## Installation

`composer install`

`composer require laravel/ui`

`php artisan ui vue --auth`

`npm install && npm run dev`

`php artisan migrate`

## Running app

`php artisan serve`

`npm run serve`

## Running Tests

`php artisan test`
